== Capture Heatmap

Draw Heatmap Layer of captured portals on Google Maps.

=== Configuration

<code>cp .env.example .env</code>

and edit +.env+

=== Deployment instructions

<code>foreman start</code>

=== Import logs

<code>foreman run ./bin/rake import_log:worker</code>
